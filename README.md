<!-- USAGE EXAMPLES -->
## Usage

```sh
$python word-parifier.py
```
You'll be asked to specify:
- The number of words you'd like in each set.
- The number of sets you'd like to generate
- A filename for the output .csv. (e.g.{banana}) ==> banana.csv
